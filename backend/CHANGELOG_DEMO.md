# PTM Now — What's new

Hand-written, teacher/parent-facing release notes for the demo panel.
Edit this file by hand: each `## heading` is a section, each `- line` is a bullet.
Blank lines and this intro paragraph are ignored by the parser.

## Latest
- Auto-schedule now animates meetings into your cart one by one, with an optional pop sound
- Parents can save private notes on each meeting, searchable from a Notes tab
- Teachers get the same private per-meeting notes, separate from the parent's

## Recent
- "Hasn't booked yet" list so admins can see which families still need to book
- Bulk actions for teachers/admins: block, unblock, or remove several slots at once
- Clearer messages when a slot can't be booked (already taken, blocked, or a clash)

## Earlier
- One-tap auto-schedule builds a conflict-free plan across all your teachers
- Book every teacher from a single grid, then confirm them all together
- Real email login codes (OTP) replace the old placeholder code
